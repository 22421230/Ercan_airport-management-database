## Project Overview

This project implements a relational database management system (RDBMS) for **Ercan Airport**. The system stores and manages information about:

- Airplanes and their models
- Hangars and occupancy history (check‑in / check‑out)
- Test definitions and testing events (which technician tested which airplane, when, hours spent, score)
- Employees (technicians, controllers, pilots) and their union membership
- Pilot licenses and medical exam expiry
- Maintenance records
- Flight schedules
- Suppliers

The design exceeds the minimal project requirements by adding **four extra tables** (`PILOT`, `MAINTENANCE_RECORD`, `FLIGHT_SCHEDULE`, `SUPPLIER`) and includes advanced SQL features (stored procedure, view, complex joins, aggregations).
